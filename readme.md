### 选项：

  `--name, -n`   组件名                                                   [字符串]

  `--mname, -m`  模块名                                                   [字符串]

  `-h, --help`   显示帮助信息                                               [布尔]

### 示例：

  `onion --name test-abc --mname test1`  创建一个组件
  
### 安装：
   `yarn add onion-cli -g`
